    Queues are a type of data structure that follow the FIFO (First In First Out) principle.

    Queue<Integer> queue = []
queue << 1
queue << 2
queue << 3

    Queue<Integer> queue = new LinkedList<>();
queue.add(1);
queue.add(2);
queue.add(3);

    val queue = LinkedList<Int>()
queue += 1
queue += 2
queue += 3

    from collections import deque

queue = deque(1, 2, 3)
